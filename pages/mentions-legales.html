2. Objet et nature du service

Bazzart est un prototype de marketplace B2C (Business-to-Consumer) destine a la demonstration educative et pedagogique. Le service comporte les fonctionnalites suivantes:

  • Presentation d une interface de marketplace avec vitrine acheteur
  • Simulation de parcours utilisateur (inscription, navigation, commande)
  • Demonstration de fonctionnalites vendeur (dashboard BI, gestion produits)
  • Illustrations de cas d usage e-commerce

Important: Aucune transaction financiere reelle, aucun traitement de paiement veritable, et aucune execution logistique ne sont effectues. C est une simulation a fins educatives.

3. Hebergement et infrastructure

Le prototype Bazzart est concu pour une execution locale uniquement:

  • Execution locale via un serveur de development (Live Server, VSCode, etc.)
  • Stockage des donnees localement dans le navigateur (localStorage)
  • Pas d infrastructure cloud, pas de bases de donnees distantes
  • Pas de synchronisation serveur/client

Le prototype ne dispose pas d hebergement permanent. Les donnees entrees sont locales a la session de navigation.
