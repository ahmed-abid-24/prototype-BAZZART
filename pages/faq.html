Questions frequentes - Acheteurs

1. Comment passer une commande sur Bazzart?

Pour commander, naviguez dans la Marketplace, selectionnez les produits qui vous interessent, ajoutez-les a votre panier, et completez le processus de checkout. Vous devrez vous identifier ou creer un compte.

2. Comment suivre ma commande apres l achat?

Une fois votre commande confirmee, vous recevez un numero de commande. Vous pouvez consulter l etat de votre commande (Preparation, Expedition, Livraison) depuis votre historique dans « Mon Compte ».

3. Quels modes de paiement sont acceptes?

Le prototype accepte les modes de paiement suivants: Carte bancaire (Visa/Mastercard), Virement bancaire, Paiement a la livraison. Tous les processus sont simules a des fins pedagogiques.

4. Quel est le delai de livraison?

Les delais de livraison varient selon le vendeur et la localisation, generalement entre 24h et 72h. Ces delais sont a titre indicatif dans le cadre du prototype.

5. Comment utiliser un code promo ou une reduction?

Vous pouvez appliquer un code promo au moment du checkout. Entrez le code promotionnel dans le champ « Code Promo » et validez. La reduction s appliquera immediatement a votre commande.

6. Puis-je retourner un produit? Quel est le delai?

Vous disposez d une periode de 7 jours a partir de la livraison pour demander un retour ou un echange. Connectez-vous a votre compte et selectionnez « Demander un retour » sur la commande concarmee.

7. Comment laisser un avis sur un produit ou le vendeur?

Une fois votre commande livree, une section « Evaluer » apparait avec le produit. Vous pouvez noter entre 1 et 5 etoiles et laisser un commentaire detaille sur votre experience.

8. Comment modifier mon adresse de livraison?

Vous pouvez modifier votre adresse de livraison dans « Mon Compte » > « Adresse ». Cette adresse s appliquera a votre prochaine commande. Pour modifier une commande deja placee, contactez le support.

9. Comment supprimer mon compte?

Pour supprimer votre compte, rendez-vous dans « Mon Compte » > « Securite » > « Supprimer mon compte ». Attention: cette action est irreversible.

10. Comment contacter le support client?

Vous pouvez nous contacter via l email support@bazzart.tn, par telephone au +216 70 000 000, ou via la page de contact. Nos equipes vous repondront sous 24h.

Questions frequentes - Vendeurs

1. Comment m inscrire comme vendeur sur Bazzart?

Cliquez sur « Devenir Vendeur » et completez le formulaire d inscription (4 etapes): informations personnelles, details du magasin, localisation, et ajout du premier produit. Votre compte sera actif immediatement (demo).

2. Quelles sont les exigences pour vendre sur Bazzart?

Vous devez avoir une adresse email valide, un numeros de telephone, et produire au minimum 1 produit. Vous devez egalement accepter les Conditions Generales de Vente de Bazzart.

3. Comment ajouter des produits a mon magasin?

Depuis votre Dashboard Vendeur, rendez-vous dans « Gestion Produits » et cliquez sur « Ajouter un produit ». Remplissez les details (nom, description, prix, images, categorie, stock).

4. Comment sont distribues les revenus et les commissions?

Bazzart preleve une commission de 10% sur chaque vente. Le paiement des revenus s effectue mensuellement via virement vers votre compte bancaire (simulation).

5. Comment repondre aux avis clients?

Dans votre Dashboard, allez dans « Gestion Avis » et selectionnez un avis. Vous pouvez ajouter une reponse de maksimum 500 caracteres. Votre reponse sera visible publiquement.

6. Comment gerer les retours de clients?

Les demandes de retour apparaissent dans votre section « Gestion Retours ». Vous pouvez accepter ou refuser le retour en justifiant le motif.

7. Comment puis-je consulter mes statistiques et mon chiffre d affaires?

Votre Dashboard affiche des graphiques detailles: ventes par categorie, revenus mensuels, trafic, nombre de commandes, et taux de conversion.

8. Comment modifier les informations de mon magasin?

Allez dans « Parametres Boutique » pour mettre a jour votre logo, slogan, description, adresse et coordonnees de contact. Les modifications prennent effet immediatement.

Questions techniques

1. Bazzart est-il un vrai service?

Non. Bazzart est un prototype academique cree dans le cadre du module Entrepreneuriat a l IHEC Carthage. Il s agit d une demonstration educative sans transactions reelles.

2. Mes donnees personnelles sont-elles securisees?

Les donnees saisies sont stockees localement dans votre navigateur (localStorage) et ne sont pas transmises a des serveurs externes. Aucun traitement réel n est effectue.

3. Puis-je utiliser Bazzart sur mobile?

Oui, le prototype est entierement responsive et fonctionne sur tous les appareils (desktop, tablette, smartphone) avec une interface adaptee.

4. Que se passe-t-il si je vide mes cookies et donnees de navigateur?

Vos donnees (panier, compte, commandes) seront effacees. Vous devrez vous reconnecter et recommencer votre shopping. Pensez a faire sauvegarder vos donnees si necessaire.
